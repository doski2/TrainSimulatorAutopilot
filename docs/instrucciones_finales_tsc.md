# Instrucciones finales TSC (Train Simulator Classic)

Instrucciones para configurar TSC y Raildriver y ejecutar pruebas con
`test_tsc_real.py`.

## Pasos

1. En TSC (RailWorks), navega al menú principal y selecciona `Drive`.
2. Elige escenario (recomendado: Clinchfield - DTG) y carga la cabina de la
locomotora.
3. Verifica que Raildriver muestre `Connected to RailWorks`.
4. Ejecuta: `python test_tsc_real.py` para iniciar las pruebas de integración.

⚠️ Notas: Mantén la ventana del simulador activa, verifica firewall/antivirus y
permisos de escritura en plugins.
